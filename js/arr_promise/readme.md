- 有一堆的异步任务要执行
- 每一项是一个promise
- Promise.all() 本身也是一个promise
  - 所有项都解决了，都成功，Promise.all()才成功
  - 只要有一个失败了，整个全部失败
  - Promise.all()成功后，返回的是一个数组，按照原promise的下标存放
    - 数组的每一项是每一个promise的resolve的值
- Promise.all() 是一个静态方法，不是实例方法
