## JS 数据类型
- primitive 简单数据类型 栈内存

  - string number undefined null boolean bigint symbol

- 复杂数据类型 栈内存（连续）（地址的引用） + 堆内存（对象的内容）
  - 引用式赋值
  - object

- string、numberic、null、undefined、boolean、symbol
