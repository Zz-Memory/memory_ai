  - V8引擎（Chrome 心脏，负责解析和执行代码）
  - 编译阶段
    - 代码执行环境
        currentVariable{

        }

      - 全局作用域
      - 函数作用域
      - 块级作用域
    - 作用域链 变量查找的路径
      - 当前作用域 -> 父作用域... -> 全局作用域
    - 嵌套关系
      - 冒泡查找
